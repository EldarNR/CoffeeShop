## Быстрая установка

```bash
# 1. Клонировать репозиторий
git clone https://github.com/EldarNR/CoffeeShop.git
# 2. Перейти в папку проекта
cd coffee-shop
# 3. Установить зависимости
npm install
# 4. Запустить проект в режиме разработки
npm run dev

После запуска проект будет доступен по адресу: http://localhost:3000

Технология	Описание
Next.js (Pages Router)	Фреймворк для React с серверным рендерингом
TypeScript	Статическая типизация для повышения надёжности кода
Zustand	Лёгкое и удобное состояние для управления данными (store)
Tailwind CSS	Утилитарный CSS-фреймворк для быстрой стилизации интерфейсов
Компонентный подход	Логичная структура проекта, разделённая на переиспользуемые UI-компоненты
